# Security Policy 🛡️

## Supported Versions

We actively maintain the latest portable release of the AS3 Professional Suites.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

As this project consists of offline, single-file HTML applications, security risks are primarily related to calculation logic and data integrity.

**How to report:**
If you find a security-critical bug or a severe error in the calculation logic (which could lead to professional liability issues), please report it via:

1. **GitHub Issues**: Open a new issue with the label `security` or `bug-critical`.
2. **Private Disclosure**: If the issue is sensitive, please contact the maintainers via the contact details provided in the main README or profile.

## Our Responsibility
We aim to fix critical logic errors or security vulnerabilities within 48 hours of confirmation. Since this is an offline tool, updates will be provided by releasing new versions of the portable HTML files.

## Privacy Notice
The AS3 Professional Suites operate **100% offline**. They do not transmit data to any server. Your inputs and calculations remain entirely within your browser's session.
