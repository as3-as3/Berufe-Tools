## 🚀 Vision
Dieses Projekt digitalisiert das fachspezifische Wissen deutscher Berufe. Weg von komplexen Software-Abos, hin zu **robusten, offline-fähigen Werkzeugen**, die direkt am Einsatzort (Baustelle, Kanzlei, Klinik) funktionieren.

## 💎 Features jeder Suite
- **Single-File-Architecture**: Alle Skripte, Styles und Logiken sind in einer einzigen HTML-Datei eingebettet.
- **Klinische Präzision**: Berechnungen basieren auf aktuellen DIN/EN/VDE Normen und gesetzlichen Vorgaben (BGB, StVO, BKatV, DüV).
- **Premium Design**: Modernstes Interface mit interaktiven SVGs, Dark-Mode Support und intuitiver UX.
- **100% Datenschutz**: Keine Server-Kommunikation. Alle Daten bleiben lokal in deinem Browser.

## 🛠️ Nutzung
1. Navigiere zum [Master Hub (index.html)](index.html).
2. Wähle deinen Beruf.
3. Lade die entsprechende `_PORTABLE.html` Datei herunter.
4. Öffne sie in einem beliebigen Browser (Chrome, Firefox, Safari, Edge).
